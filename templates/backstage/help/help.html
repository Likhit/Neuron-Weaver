Help

Neuron weaver is a GUI based tool which lets you build, run and train neural networks easily. The following documentation should help you get started. It will be helpful to also open this application in another tab and try out the actions while reading the documentation.

Build a neural network

The first step is to build a neural network according to our desired architecture. There are 3 tools present in the BUILD tab to help us do so; namely the Neurons(currently only the Hebb neuron implemented), the Connection and the Eraser.

Add a neuron:

  • To start building a network select the Hebb neuron from the ribbon at the top.
  • On moving the mouse pointer over the canvas(page) at the center you will see that the mouse cursor has turned into cross-hair.
  • Click anywhere on the canvas to place a neuron.
  • The neuron should now be placed on the canvas. You can place more neurons by clicking on the canvas again(preferably at a different spot).
  • Each neuron has 2 components displayed on the screen.
    • On the top, next to the #, is the neuron's code i.e. a number to uniquely represent the neuron.
    • At the bottom of the neuron, next to Î¸, is the threshold value of the neuron. The Hebb neuron follows the simple threshold function which fires if the net input is >= the threshold value of the neuron.
  • Once done, deselect the Hebb neuron from the ribbon.
  • You van add more neurons anytime by reselecting the Hebb neuron tool.

Drag a neuron to a new location:

  • Click on the neuron already placed on the canvas and hold the mouse down.
  • With the mouse held down, drag the mouse pointer across the canvas to the desired location.
  • Place the neuron in this new location by letting go the mouse pointer.

Add a connection between two neurons:

  • Select the Connection tool from the ribbon.
  • On the canvas, click on the boundary of the neuron you want to make a connection from.
  • While holding the mouse button down, move the mouse pointer across the neuron to connect to.
  • At the boundary of the second neuron, leave the mouse button.
  • If a connection was made successfully, the connection will turn black color. If not, erase the connection and make a new connection. Make sure to start and end the connection from near the boundaries of the neurons.
  • You can continue adding connections as long as the Connection tool is selected.
  • Near this new connection will be a number representing the weight of this connection. You can drag-drop the weight number to any desired location.
  • Click again on the Connection tool from the ribbon to deselect it.

Erase a neuron or connection:

  • Select the Eraser from the ribbon.
  • To erase a neuron, just click on the neuron.
  • To erase a connection, click on the weight of the connection.
  • Click again on the Eraser to deselect it.

Setting a neuron's threshold:

  • Click on the neuron to set its threshold.
  • A form should have popped up at the bottom of the screen.
  • Enter the new threshold value in the form field and click set(alternatively press return).

Setting a connection's weight:

  • Click on the weight of the connection.
  • A from should have popped up at the bottom of the screen.
  • Enter the new weight of the connection and press return.

Run a neural network

A neural network which has been created can be run/simulated on any specified input. Option to simulate a neural network can be found in the RUN tab. On switching from the BUILD tab to the RUN tab, the neural network created is processed and the input and output nodes of the network are found out.

The input nodes can be identified by the arrow to their left. Each input node will just pass in any input fed to it without processing, this is done by setting its threshold to 1. Therefore, be careful while changing the threshold of an input node. The output nodes can be identified by the concentric circles. An output node behaves like a normal neuron but is special since it returns the final output of our problem. Its denotation with concentric circles is purely decorative.

The inverse i.e. the dismantling of the neural network into neurons with no special meaning occurs when switching from the RUN tab to the BUILD tab.

Feeding the network inputs:

  • Click on the Set Inputs button on the ribbon(in the RUN tab).
  • A form should have popped up at the bottom of the screen.
  • Enter the inputs for the input neurons(neurons are identified by their code i.e #) in the table.
  • Each row in the table represents one input set. More inputs sets can be added using the Add Input Set button.
  • At the top of the form is a field to specify the rate at which each input set is fed to the neural network. The value to be specified here will normally be the length of the longest path(without loops) from an input neuron to an output neuron.
  • Finish off by clicking on Set(alternatively press return).

Running the network:

  • After feeding the network inputs it can be run using the Run button in the ribbon.
  • This will simulate the network on the supplied input sets. Each input set will be fed every n time units where n is the number specified in the Set Input form's "Feed inputs every n time units". Each time unit is 1 second real time.
